# Ruvyxa branding assets

This directory is the framework-owned source of truth for Ruvyxa branding assets.

- `ruvyxa.png` is the canonical framework logo.
- `templates/minimal/public/ruvyxa.png` and `examples/demo/public/ruvyxa.png` are runtime copies
  used by generated/demo applications and the framework error page at `/ruvyxa.png`.
- When the logo changes, update the canonical asset first, then synchronize the runtime copies.
